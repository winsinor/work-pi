Location

Used for weather, AQI, and NWS alerts. Find lat/lon at latlong.net.

Addresses

Full street addresses for TomTom routing. Geocoded once at startup.

If set, adds a second commute route: Work → Waypoint → Home.

API Keys

Free tier at developer.tomtom.com — needed for geocoding and routing.

No-key APIs

Weather, AQI, and NWS alerts use Open-Meteo and api.weather.gov — both free, no key needed.

Outlook Calendar

In Outlook Web: Settings → View all → Calendar → Shared calendars → Publish calendar → copy the ICS link.
Leave blank to disable calendar and WFH/OOO detection.

WFH / OOO Detection

The app scans today's all-day calendar events. If a title contains any of these keywords the display switches to the matching mode.

e.g. wfh, working from home

e.g. ooo, out of office, pto
